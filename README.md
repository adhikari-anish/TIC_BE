# Tourism Information Center (TIC) Backend

This is a backend repo for our microservices implementation of TIC.

## Prerequisite

* Docker version 25.0.2, build 29cf629

## Running the project

* docker-compose build
* docker-compose up


## Built With

* Express.js
* mongodb
* docker
* RabbitMQ
* nginx
* AWS
